# My resume as an npm package!

Exciting, isn't it? 🎉

Next steps:
1. Install the npm package by runnning the code:
 ```
 npm install -g omer-resume
 
 ```
 
2. After installation is completed type 
```
omer-resume
```
and press enter to know me 😄
